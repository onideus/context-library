# Contributing to Context Library

Thank you for your interest in contributing to Context Library.

## Getting Started

1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes
4. Run `npm run typecheck` to verify
5. Open a pull request

## Development Setup

```bash
git clone https://github.com/onideus/context-library.git
cd context-library
cp .env.example .env
docker compose up -d
```

## Guidelines

- Keep PRs focused on a single change
- Include relevant context in PR descriptions
- Follow existing code style and conventions
- Do not commit secrets, personal data, or environment-specific configuration
